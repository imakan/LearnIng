# 使用Chrome Performance归档组件

+ 在项目地址栏内添加查询字符串 ?react_perf（例如， http://localhost:3000/?react_perf）。

+ 打开Chrome开发工具Performance 标签页点击Record.

+ 执行你想要分析的动作。不要记录超过20s，不然Chrome可能会挂起。

+ 停止记录。

+ React事件将会被归类在 User Timing标签下。
